# Fika Week Scheduler

This is a simple static HTML app that shows the next fika responsible and lets you swap future fika weeks.

## Local preview

Open `test.html` in your browser directly, or run a simple static server:

- Python 3:
  ```powershell
  python -m http.server 8000
  ```
  Then open `http://localhost:8000/test.html`.

## Publish on GitHub Pages

1. Create a GitHub repository and push this file.
2. In the repo settings, enable **GitHub Pages** from the `main` branch and root folder.
3. Your app will be available at `https://babsan321.github.io/fika_responsibility/index.html`.

## Publish on Netlify

1. Go to https://app.netlify.com.
2. Drag and drop the project folder to deploy.
3. Or connect the GitHub repo and deploy the `main` branch.

## Publish on Vercel

1. Go to https://vercel.com.
2. Import your GitHub repository.
3. Deploy the project as a static site.

## Notes

- The app is entirely static HTML/CSS/JavaScript.
- Any static hosting service can serve it publicly.
